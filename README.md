# Basic HTML Website

A simple personal portfolio page built with pure HTML — no CSS, no JavaScript. Focuses on semantic markup and proper document structure.

Part of the [roadmap.sh](https://roadmap.sh/projects/basic-html-website) frontend project series.

## Run

Clone the repo and open `index.html` in your browser — that's it.
